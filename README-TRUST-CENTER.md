# Vordali Trust Center Upload Package

Upload every file in this package to the root of the existing `vordali-website` GitHub repository.

The included `script.js` replaces the existing file and preserves the current homepage behavior while adding Trust Center and policy links to the homepage footer.

Do not delete the existing `assets` folder. These pages reuse:
- assets/vordali-mark.webp
- assets/favicon.png

Suggested commit:
Trust Center - Privacy Terms SMS and Security

After deployment use these URLs for Twilio:
- https://vordali.com/privacy.html
- https://vordali.com/terms.html
- https://vordali.com/sms-terms.html

These are practical startup templates tailored to Vordali's current design and are not legal advice. Obtain attorney review before onboarding paying businesses or operating at scale.
